# Roku Flashback

This website is my 'Roku Flashback' app, this is in conjunction with my program Interactive Media Design at Fanshawe College. 

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes. See deployment for notes on how to deploy the project on a live system.

### Prerequisites

You must have the following programs installed: 

* MAMP/ WAMP

### Installing

You will need to install the following in order to run the website:

To see the sign up function you need to install the .sql file from the 'data' folder, this can be done in the Terminal or through phpMyAdmin. Name the database 'db_roku'.

## Deployment

* Once everything is installed, now start the MAMP/WAMP server

* Then in the browser of your choice go to: localhost:(your MAMP/WAMP server number)
